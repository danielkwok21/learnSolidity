## Remix MVP contracts

These are meant to be run in https://remix.ethereum.org/

## Steps
1. Pick a contract (*.sol file)
2. Copy content
3. Go to https://remix.ethereum.org/
4. Create a new file in ``contracts/<contract-name>.sol``
5. Paste content
